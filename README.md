# Aral

This is a personal experiment in knowledge extraction and synthesis. The
pipeline is intentionally light-weight: input is plain-text files, the database
is a set of JSON files, and output is in Markdown.

The knowledge domain for this experiment is a long-running D&D game I’m playing
with my brother Rogers and friend Thain. It’s Thain’s game, set in an original
world he create4d called Aral. Rogers takes moment-by-moment notes of our
adventures in Discord. Those session notes are then grist to the wiki mill.
